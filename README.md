# Press
## 简要介绍
一个纯净的记录自己想法的地方

## 目的
基于 React 技术栈的个人项目，同时也用来实践各种技术

## 技术选型
Webpack + Babel + Typescript

## 特别之处
使用了 Babel 7 ，因此使用了 Babel 来解析 Typescript ，抛弃了 ts-loader ，用来缓解 ts-loader 速度较慢的问题
